How PageIndex RAG Works

PageIndex is a vectorless, reasoning-based RAG system inspired by AlphaGo:

  1. Builds a Tree Index — hierarchical TOC preserving document structure.
  2. Reasons Through the Tree — LLM navigates using reasoning, not similarity.
  3. Retrieves Precise Context — exact pages and sections with explainability.
  4. Generates Answers — accurate, well-cited responses from retrieved context.

Achieves 98.7% accuracy on FinanceBench benchmark.
